Algorithms in go
==================

## To Run

1. [Install Go](https://golang.org/doc/install)
2. Clone the repo to `$GOPATH/src/algorithms-in-go`
    ```
    git clone git@github.com:songe/algorithms-in-go.git $GOPATH/src/algorithms-in-go
    ```
3. Run the tests
    ```
    go test algorithms-in-go
    ```
